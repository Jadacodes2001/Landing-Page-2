# Manipulating the DOM Landing Page

This project demonstrates the creation of a responsive landing page using JavaScript and DOM manipulation. The landing page includes a dynamically generated navigation menu, active section highlighting, and smooth scrolling behavior. It's designed to be usable across modern desktop, tablet, and phone browsers.

## Usage

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Scroll through the sections to see the active section highlighting and smooth scrolling in action.
4. Click on navigation items to scroll to the corresponding sections.

## Dependencies

- This project relies on HTML, CSS, and JavaScript to create the landing page.
- It uses the Google Fonts API to load fonts for the page.

## Project Structure

- `index.html`: Contains the HTML structure of the landing page.
- `css/styles.css`: Includes the CSS styles for the landing page.
- `app.js`: Contains the JavaScript code for creating the dynamic navigation, smooth scrolling, and active section highlighting.

## Features

- Responsive layout for modern desktop, tablet, and phone browsers.
- Dynamically generated navigation menu based on page sections.
- Active section highlighting as users scroll through the page.
- Smooth scrolling behavior when clicking on navigation items.
